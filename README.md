# miya
毕设：Miya电商平台


# About
此项目是 Spring Cloud + Vue/Jsp + Kafka + Solr + FastDFS 构建的B2C电商系统，所有的数据都是从服务器实时获取的真实数据，具有真实的注册、登陆、管理数据等功能。
